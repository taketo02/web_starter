# web_starter

データサイエンス実践演習 A 第6回 実習用のスターターリポジトリです。

## 内容

簡単な Web ページ（HTML + CSS + JavaScript）の最小構成。

- `index.html` ── ページ本体（自己紹介テンプレ）
- `style.css` ── 見た目の装飾
- `script.js` ── 動きを足す JavaScript

## ブラウザで開く

エクスプローラから `index.html` をダブルクリック。
WSL2 の場合は `explorer.exe .` でディレクトリを開いてから `index.html` をダブルクリック。

## 自分のものに 書き換える

`index.html` の `<!-- ここを 書き換える -->` の部分を、自分の情報に差し替えてください。

```bash
# 編集したら add → commit → push
$ git add .
$ git commit -m "自己紹介を 書き換えた"
$ git push
```

## このリポジトリの目的

GitHub に push する 練習用です。中身は シンプルですが、HTML / CSS / JavaScript の おさらいに なります。
